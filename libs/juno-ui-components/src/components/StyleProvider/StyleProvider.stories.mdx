# StyleProvider

This component embeds the styles of this design system in an element depending on the value
of the property "stylesWrapper".

This property can contain the following values:

- **head** add style tag with css in the head element
- **inline** insert a style tag at the point where the SytleProvider is integrated.
- **shadowRoot** create a shadow root element and append it to the parent element of the StyleProvider.
  All styles are placed within this shadow root. The ShadowRoot component is used for this.
