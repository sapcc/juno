# ShadowRoot

Creates a shadow dom element, adds it to the parent node, and inserts all children elements inside itself.
ShadowRoot allows css styles to be isolated from the rest of the DOM. Choosing "closed" for **mode** prevents
the styles from being inherited from the parent node.

The component accepts three parameters:

- **mode**: "open" (default) or "closed"
- **delegatesFocus**: true or false (default),
- **styles**: css styles as a string,
